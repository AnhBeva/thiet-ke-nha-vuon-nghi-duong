# Quy Ước Đặt Tên File Và Tổ Chức Dự Án

## Cấu trúc thư mục thực hành

Tạo một thư mục riêng cho bài tập dự án:

```text
01_SketchUp
02_D5
03_Render
04_Video
05_Tham_khao
06_Checklist
07_Xuat_ban
```

## Quy ước đặt tên

Tên file nên có ngày, nội dung và phiên bản:

```text
2026-05-14_nha-vuon_tong-mat-bang_v01.skp
2026-05-14_nha-vuon_d5_v01.drs
2026-05-14_render_cong-vao_v01.jpg
```

## Nguyên tắc lưu phiên bản

- Không ghi đè file đã dùng để ra quyết định.
- Mỗi lần đổi lớn tạo phiên bản mới.
- File nháp để trong thư mục riêng hoặc ghi rõ `nhap`.
- File gửi người khác phải dùng tên dễ hiểu, không dùng `final_final_2`.
